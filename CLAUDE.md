# Agentify - Meta Tool for Agent-Native Product Transformation

## Project Vision

Agentify 是一个 AI-Native 的 meta tool，帮助 AI Agent 理解和转化传统产品为 Agent-Native 产品。
它是一个"产品范式转换器"——将传统 GUI/SaaS 产品能力转化为 MCP Server、API、Skills 等 Agent 可消费的接口。
